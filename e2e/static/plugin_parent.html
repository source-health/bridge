Source Simple Plugin Demo

About

This is a simple demo/testing scaffold that loads demo_plugin.html and emulates the Source application, responding to the initial 'hello' message, and passing context and tokens to the iframe.

The demo_plugin.html file demonstrates how a frame plugin works within the Source web application. The plugin javascript does the following:

  1. Subscribe to context updates from the parent window, which will cause the context received to be displayed in the iframe.
  2. Call SourceBridge.init() to kick off the required handshake with the parent window.
